## 📌 О проекте

**ТЗ Анализатор** — веб-платформа, которая использует генеративный ИИ (Claude от Anthropic) для автоматического анализа технических заданий, выявления рисков и генерации профессиональных ТЗ с нуля.

Решает ключевую проблему рынка: **низкое качество ТЗ** в государственных и коммерческих закупках, которое приводит к срывам проектов и судебным разбирательствам.

## 🚀 Быстрый старт

### Вариант 1 — Открыть напрямую (без сервера)

1. Скачай или клонируй репозиторий
2. Открой `index.html` в браузере
3. Введи API-ключ от [Anthropic](https://console.anthropic.com/settings/keys)
4. Готово!

### Вариант 2 — С сервером (Node.js)

```bash
# Установи зависимости
npm install

# Создай файл .env (скопируй из .env.example)
cp .env.example .env
# Укажи свой API-ключ в .env

# Запусти сервер
npm start
```

Открой браузер: [http://localhost:3000](http://localhost:3000)

### Вариант 3 — Режим разработки (авто-перезапуск)

```bash
npm run dev
```

## 🛠️ Технологии

- **Frontend:** HTML5, CSS3, Vanilla JavaScript (без фреймворков)
- **AI:** Anthropic Claude API (claude-sonnet)
- **Backend (опционально):** Node.js + Express
- **Парсинг документов:** Mammoth.js (DOCX)
- **Безопасность:** PBKDF2 (100 000 итераций, SHA-256), CSP, сессионные токены
- **Хранилище:** localStorage (браузер)

## 🔐 Безопасность

- Пароли хешируются через **PBKDF2** (100 000 итераций, SHA-256, уникальная соль)
- Сессионные токены с **TTL 8 часов** и автовыходом при неактивности
- Защита от брутфорса: **блокировка на 15 минут** после 5 неверных попыток
- **Content Security Policy (CSP)** — защита от XSS-атак
- API-ключ хранится только в браузере пользователя и никуда не передаётся
